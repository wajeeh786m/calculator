# 🧮 Premium Calculator

A modern, responsive calculator application built using **HTML**, **CSS**, and **Vanilla JavaScript**. Designed with a premium dark UI, smooth animations, and intuitive interactions, this project demonstrates clean frontend development, responsive design, and JavaScript application logic.

- Modern premium dark interface
- Fully responsive design

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/wajeeh786m/calculator.git
```

Open the project folder:

```bash
cd calculator
```

Run the application by opening:

```
index.html
```

in your preferred web browser.
